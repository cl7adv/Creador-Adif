ADIF PROTOCOL

Dominando el intercambio de datos en la Radioafición

01.

La Estructura de Datos

El formato ADIF no es un simple texto. Es el lenguaje universal entre loggers. Cada campo se define por su longitud y nombre.

<CALL:5>EA1ABC <QSO_DATE:8>20260103 <EOR>
02.

Cabecera Obligatoria

Todo archivo profesional debe comenzar con una cabecera que defina la versión del estándar.

ADIF Export from MyEpicStation v1.0
<ADIF_VER:5>3.1.4
<EOH>
03.

El Registro (QSO)

Cada contacto termina con la etiqueta <EOR> (End Of Record). Asegúrate de incluir frecuencia, modo y reporte RST.
